$^3$He library

Vladislav Zavjalov

This library provides constants and functions for various $^3$He properties.
Interfaces for Fortran, C, Matlab, Octave and command line are available.
The source code is availabe from https://github.com/slazav/he3lib.
I would like to recieve your suggestions, corrections etc. via e-mail slazav@altlinux.org
